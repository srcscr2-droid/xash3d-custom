# xash3d Android working skeleton

This archive contains a minimal Android Studio project that builds a real APK.

What is included:
- `app/src/main/java/.../MainActivity.kt`
- `app/src/main/AndroidManifest.xml`
- `app/src/main/res/*`
- Gradle build files

What is not included:
- Xash3D engine sources
- HLSDK
- NanoGL
- VGUI

Open the root folder in Android Studio and sync the project.
